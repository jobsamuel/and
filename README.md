# and

A demo single page application built with AngularJS.

# usage

### Pre-requisites

 - [NodeJS](http://nodejs.org/)
 - [Bower](http://bower.io/)
 - [noem](https://github.com/jobsamuel/noem) —or any other RESTful API.

### Setup

```
$ git clone git@github.com:jobsamuel/and.git
$ npm install
$ bower install
$ node server.js
```
